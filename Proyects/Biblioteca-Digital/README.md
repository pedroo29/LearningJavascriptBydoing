# Biblioteca Digital

Aplicación demo de biblioteca digital, permite prestado y devolución de libros, base de datos de usuarios y libros con archivos mock, permite también la creación de nuevos usuarios y libros.

## Implementación y aprendizaje

Al hacer la aplicación he usado el patron de diseño estructural facade, donde he aprendido un poco como funciona, he usado y aprendido sobre los modelos Anemic Domain Models, que son modelos que únicamente sirven como contenedores y he implementado la lógica de prestamo de libros en una clase servicio (LoanService), la clase Library es la fachada de la biblioteca donde se usa las implementaciones de todo el sistema que hay detrás.
